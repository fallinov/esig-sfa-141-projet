# Projet évaluatif - Module C141

Site de présentation du projet évaluatif pour le module C141 (Développer des applications web adaptatives) à l'ESIG.

## Contenu

- **Objectifs** : Architecture composants, gestion d'état, connexion API, design responsive
- **À rendre** : Application Vue.js + Vuetify avec API externe
- **Exigences** : Vue.js 3, Pinia, Axios, Vuetify, fonctionnalités requises
- **Planification** : 6 semaines avec jalons hebdomadaires
- **Évaluation** : 40% projet + 60% défense orale
- **Défense orale** : 20 minutes (démo + questions théoriques + questions pratiques)
- **Support & IA** : Règles d'utilisation de l'IA
- **Checklist** : Auto-évaluation avant rendu
